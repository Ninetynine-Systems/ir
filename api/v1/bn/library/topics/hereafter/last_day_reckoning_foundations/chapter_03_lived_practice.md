# জীবনে প্রয়োগ

## মূল সূত্র

- সূরা আয-যালযালাহ ৯৯:৭-৮
- সূরা আল-আম্বিয়া ২১:৪৭

হিসাবের বিশ্বাস ছোট আমলকেও অর্থপূর্ণ করে। গোপন দয়া, সংযত কথা, নির্জন তাওবা, বা থেমে যাওয়া গুনাহ - আল্লাহর কাছে হারিয়ে যায় না। আবার ছোট অন্যায়ও শুধু মানুষ দেখেনি বলে মুছে যায় না।

বাস্তব পথ হলো ভারসাম্য: আল্লাহর রহমতের আশা রাখা, জবাবদিহিকে ভয় করা, অন্যায় ঠিক করা, এবং বারবার ফিরে আসা। শেষ দিনের স্মরণ অন্তরকে দেরি হওয়ার আগেই জাগানোর জন্য; তাওবার দরজা খোলা থাকা অবস্থায় তাকে হতাশ করার জন্য নয়।
