# মূল আয়াতগুলো

## মূল সূত্র

- সূরা আল-আম্বিয়া ২১:৪৭
- সূরা আল-ইনশিকাক ৮৪:৭-১২
- সূরা আল-কারি'আহ ১০১:৬-১১

কুরআন ন্যায়সঙ্গত মীযান, লিখিত আমলনামা, এবং এমন পরিণতির কথা বলে যা কেউ উপেক্ষা করতে পারবে না। সূরা আল-আম্বিয়া ২১:৪৭ ন্যায়বিচারের দাঁড়িপাল্লা স্থাপনের প্রতিশ্রুতি দেয়। সূরা আল-ইনশিকাক ডান হাতে আমলনামা পাওয়া ব্যক্তির সঙ্গে পেছন দিক থেকে আমলনামা পাওয়া ব্যক্তির পার্থক্য দেখায়। সূরা আল-কারি'আহ আমলের ভারসাম্যকে সরাসরি সতর্কতায় পরিণত করে।

এসব আয়াত বিমূর্ত ধারণা নয়। দিনটি আসার আগেই এগুলো বিবেককে প্রশিক্ষণ দেয়।
