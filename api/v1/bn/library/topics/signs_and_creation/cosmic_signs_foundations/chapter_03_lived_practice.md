# জীবনে প্রয়োগ

## মূল সূত্র

- সূরা ইউনুস ১০:৫
- সূরা আল-মুলক ৬৭:৩-৫

আসমান থেকে উপকার নিতে একজন মুসলিমের অদ্ভুত দাবি দরকার হয় না। ফজর আসে সকাল নিয়ে, মাস জানা যায় চাঁদের মাধ্যমে, আর বারবার ফিরে আসা দিনগুলো বান্দাকে সালাত, কৃতজ্ঞতা, ও জবাবদিহির দিকে ডাকে।

আকাশকে দেখে অন্তর আল্লাহর সামনে ছোট হোক; অসমর্থিত দাবিতে উচ্চকণ্ঠ না হোক। সূর্যোদয়, সূর্যাস্ত, চাঁদের ধাপ, বা রাতের আকাশ দেখলে পর্যবেক্ষণকে ইবাদতে পরিণত করুন - সৃষ্টিকর্তার প্রশংসা করুন, সময়ের আমানত রক্ষা করুন, এবং মনে রাখুন, আপনার ওপরের শৃঙ্খলা এমন এক রহমত যা আপনি বানাননি।
