# মূল আয়াতগুলো

## মূল সূত্র

- সূরা ইয়াসীন ৩৬:৩৮-৪০
- সূরা আল-মুলক ৬৭:৩-৫

সূরা ইয়াসীন সূর্যের নির্ধারিত গতিপথ এবং চাঁদের নির্ধারিত ধাপের কথা বলে; তারপর জানায়, প্রত্যেকটি নিজ নিজ পথে চলে। এখানে শৃঙ্খলা, সীমা, এবং আনুগত্যের শিক্ষা আছে। সূর্য ও চাঁদ শক্তিশালী নিদর্শন, কিন্তু আল্লাহর তাকদিরের বাইরে নয়।

সূরা আল-মুলক দৃষ্টিকে স্তরে স্তরে নির্মিত আসমানের দিকে ফেরায় এবং মানুষকে বারবার তাকাতে বলে। উদ্দেশ্য কোনো গোপন কোড বা উত্তেজক দাবি নয়। উদ্দেশ্য বিনয় - যে বিশাল আসমান চোখকে বিস্মিত করে, সেটিও আল্লাহর নিয়ন্ত্রণাধীন সৃষ্টি।
