# মূল দলিল

## সূত্র-ভিত্তি

- সূরা আল-হাজ্জ 22:26-37
- সহিহ মুসলিম 1218
- জামি আত-তিরমিযি 889

সূরা আল-হজ 22:26-37 তীর্থযাত্রার চারপাশে অনেক অর্থ সংগ্রহ করে: ঘরটি উপাসনার জন্য শুদ্ধ করা হয়, লোকদেরকে হজ এর জন্য আসতে বলা হয়, এবং আচারগুলি আল্লাহ এর স্মরণে আবদ্ধ। এমনকি কোরবানিও তাকওয়া এর দিকে ফিরিয়ে দেওয়া হয়, কারণ মাংস বা রক্ত ​​আল্লাহ পর্যন্ত পৌঁছায় না, তবে উপাসকের তাকওয়া তা করে।

সুন্নাহ তীর্থযাত্রার মাধ্যাকর্ষণ ও করুণাকে স্পষ্ট করে। সহীহ মুসলিম 1218 একটি স্বীকৃত হজ অশ্লীলতা এবং দুষ্টতা মুক্ত ব্যক্তির জন্মদিনের মতো বিশুদ্ধ প্রত্যাবর্তনের সাথে সংযুক্ত করে এবং ভবিষ্যদ্বাণীমূলক নির্দেশিকাও আরাফাতে অবস্থানকে আচারের কেন্দ্রবিন্দু হিসাবে তুলে ধরে।
