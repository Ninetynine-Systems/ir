# কখন পড়বেন

## মূল সূত্র

- সূরা আল-বাকারাহ ২:২৫০

এই দু'আ জালুত ও তার বাহিনীর মুখোমুখি হওয়া মুমিনরা করেছিলেন। এতে প্রথমে ধৈর্য, তারপর দৃঢ়তা, তারপর সাহায্য চাওয়া হয়েছে; অর্থাৎ বাহ্যিক সাফল্যের জন্য আল্লাহর দেওয়া অন্তরের স্থিরতা প্রয়োজন।

চাপ, ভয়, কঠিন আনুগত্য, বা নীতিগত সাহসের প্রয়োজন হলে এটি পড়া মানানসই।

এখানে নির্দিষ্ট সংখ্যার নির্দেশ নেই। বৈধ উপায় গ্রহণ করে এবং দৃঢ়তাও আল্লাহর দান - এ কথা স্বীকার করে পড়ুন।
