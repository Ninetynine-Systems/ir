হজের পরের দিনগুলো ঘরে থাকা মানুষদেরও শিক্ষা দেয়। কবুল হওয়া আল্লাহর হাতে, তাই চাইতে থাকা প্রতিটি হৃদয় বিনয়ী থাকে।

> সূত্র: সহিহ আল-বুখারি ১৫২১; সূরা আল-বাকারাহ, ২:১২৭

উৎসের ওপর দিয়ে তাড়াহুড়া করে চলে যাবেন না। এটিকে পরের কয়েক ঘণ্টার জন্য ছোট দিশা হিসেবে পড়ুন, স্লোগান হিসেবে নয়। দিনটিকে নাটকীয় হতে হবে না; শুধু একটি জায়গা দরকার যেখানে স্মরণ আপনার প্রতিক্রিয়া বদলে দেয়।

এই স্মরণটি দিনের প্রথম অংশে একটি ছোট, আন্তরিক কাজে ঢুকতে দিন।
