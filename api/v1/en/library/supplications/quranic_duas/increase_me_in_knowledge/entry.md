# When To Use It

## Source anchors

- Surah Ta-Ha 20:114

Allah commands the Prophet ﷺ to say this dua. It teaches that knowledge is not merely collected; it is asked from Allah with humility.

It is fitting before study, while learning Quran, when seeking clarity, or whenever knowledge could become a source of hurry or pride. The request is simple: not just more information, but an increase that Allah grants and blesses.

No fixed repetition count is attached here. Recite it with humility and a desire for beneficial knowledge.
