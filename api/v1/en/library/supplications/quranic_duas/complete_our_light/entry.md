# When To Use It

## Source anchors

- Surah At-Tahrim 66:8

This Quranic plea gathers hope for the Hereafter with the need for forgiveness now. It is brief, but it asks for two immense gifts: completed light and pardon from Allah.

Use it when thinking about accountability, repentance, and the need for Allah to complete what our deeds cannot complete on their own.

No fixed repetition count is attached here. Recite it with hope, humility, and a heart that knows light is a gift from Allah.
