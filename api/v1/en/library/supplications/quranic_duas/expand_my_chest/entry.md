# When To Use It

## Source anchors

- Surah Ta-Ha 20:25-28

Musa made this plea when commanded to go to Pharaoh. The dua does not ask for performance or status. It asks for inner openness, ease in the task, and speech that carries meaning clearly.

Use it before teaching, difficult conversations, da'wah, leadership, study, or any responsibility where clarity and steadiness are needed.

No fixed repetition count is attached here. Recite it while preparing well and relying on Allah for ease.
